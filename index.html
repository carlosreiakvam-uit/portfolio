Predictions of flowertypes using k-nearest neighbour

K-nearest neighbor (KNN) is a supervised machine learning algorithm used for classification and regression. It works by finding the K "nearest" data points to a given input data point, and using those data points to make a prediction about the input data point.

By utilizing KNN I was able to predict correct iris type from unseen data
